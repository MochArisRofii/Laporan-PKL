Selama di Ubig, saya mengerjakan berbagai proyek. Berikut adalah proyek-proyek saya:

Project Siakadplus

SiakadPlus Adalah Aplikasi Dan Web Untuk Solusi Manajemen Sekolah

Yang Canggih Dan Ramah Pengguna. Ini Menyederhanakan

Komunikasi, Tugas Administratif, Dan Meningkatkan Pengalaman

Pendidikan, Memastikan Keunggulan Dalam Manajemen Pendidikan.

Botpress Cloud

Saya Mengerjakan Dan Sekaligus Belajar Botpress Cloud Dan Saya

Membuat Bot Di Botpress Cloud Yaitu Bot Movie, Bot Pelanggaran Siswa,

Bot BPJS, Bot Quiz Login Dengan OTP, Dan Bot Laporan Masyarakat

Django Pelanggaran Siswa

Proyek Django ini mencatat pelanggaran siswa dan mengirim notifikasi

otomatis melalui email dan WhatsApp menggunakan API Nobox.

Aplikasi ini mendukung banyak sekolah dengan penginputan

pelanggaran oleh guru, memastikan komunikasi yang cepat dan efektif

dengan wali siswa.

React Js Pelanggaran Siswa

Frontend proyek ini dibangun dengan React JS dan terhubung dengan

API Django untuk manajemen pelanggaran siswa. Aplikasi ini

menyediakan menu siswa, kategori, dan pelanggaran, memungkinkan

pengguna untuk mengelola data pelanggaran siswa secara efisien di

sekolah yang menggunakan aplikasi ini.
